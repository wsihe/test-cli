### 技术栈

vue + echart + element-ui + axios + webpack + ES6/7 + pug + stylus


### 项目运行

#### 注意：由于涉及到 ES6 等新属性，建议使用 node 最新稳定版

```
git clone https://github.com/wsihe/wuui-pro.git

cd wuui-pro

npm install

npm run serve

```

### 功能

- 登录/登出
- 收缩侧栏
- 标签导航

- 首页总览
- 商品管理
- 订单中心
- 数据管理
- 客户信息
- 店铺设置
- 个人中心
